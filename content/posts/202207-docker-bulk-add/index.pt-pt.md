O objetivo da Docker é criar uma experiência de produto de classe mundial para os nossos clientes. Queremos construir um produto robusto que ajude todas as equipas a alcançar os seus objetivos. Em linha com isso, tentámos simplificar o processo de integração da tua equipa no ecossistema Docker com a nossa funcionalidade de Adição em Massa de Utilizadores para as subscrições Docker Business e Docker Team.

Podes convidar a tua equipa para as suas contas carregando um ficheiro com os seus endereços de email no Docker Hub. O ficheiro CSV pode ser um ficheiro que crias especificamente para este propósito, ou um extraído de outro sistema interno. O único requisito é que o ficheiro contenha uma coluna com os endereços de email dos utilizadores que serão convidados para o Docker. Assim que o ficheiro CSV for carregado através do Docker Hub, cada membro da equipa no ficheiro receberá um convite para usar a sua conta.

<img src="/posts/202207-docker-bulk-add/image1.webp"/>

Também atualizámos a interface web do Docker Hub para adicionar múltiplos membros de uma vez. Esperamos que isto seja útil para equipas mais pequenas que podem simplesmente copiar e colar uma lista de emails diretamente na interface web e integrar todos os que precisam. Assim que a tua equipa estiver convidada, podes ver tanto os convites pendentes como os aceites através do Docker Hub.

<img src="/posts/202207-docker-bulk-add/image2.webp.webp"/>

A Adição em Massa de Utilizadores pode ser usada sem precisar de ter SSO configurado para a tua organização. Esta funcionalidade permite-te tirar o máximo partido da tua subscrição Docker Team ou Business, e simplifica muito o processo de integração.

Aprende mais sobre a funcionalidade na nossa página de documentação, e inicia sessão na tua conta Docker Hub para experimentar tu mesmo.
