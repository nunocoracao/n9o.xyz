L'objectif de Docker est de créer une expérience produit de classe mondiale pour nos clients. Nous voulons construire un produit robuste qui aidera toutes les équipes à atteindre leurs objectifs. Dans cette optique, nous avons essayé de simplifier le processus d'intégration de votre équipe dans l'écosystème Docker avec notre fonctionnalité d'Ajout Massif d'Utilisateurs pour les abonnements Docker Business et Docker Team.

Vous pouvez inviter votre équipe à leurs comptes en téléchargeant un fichier contenant leurs adresses email sur Docker Hub. Le fichier CSV peut être un fichier que vous créez spécifiquement à cet effet, ou un fichier extrait d'un autre système interne. La seule exigence est que le fichier contienne une colonne avec les adresses email des utilisateurs qui seront invités sur Docker. Une fois le fichier CSV téléchargé via Docker Hub, chaque membre de l'équipe dans le fichier recevra une invitation à utiliser son compte.

<img src="/posts/202207-docker-bulk-add/image1.webp"/>

Nous avons également mis à jour l'interface web de Docker Hub pour ajouter plusieurs membres à la fois. Nous espérons que cela sera utile pour les petites équipes qui peuvent simplement copier et coller une liste d'emails directement dans l'interface web et intégrer tous ceux dont ils ont besoin. Une fois votre équipe invitée, vous pouvez voir les invitations en attente et acceptées via Docker Hub.

<img src="/posts/202207-docker-bulk-add/image2.webp.webp"/>

L'Ajout Massif d'Utilisateurs peut être utilisé sans avoir besoin de configurer le SSO pour votre organisation. Cette fonctionnalité vous permet de tirer le meilleur parti de votre abonnement Docker Team ou Business, et elle simplifie considérablement le processus d'intégration.

En savoir plus sur la fonctionnalité sur notre page de documentation, et connectez-vous à votre compte Docker Hub pour l'essayer vous-même.
