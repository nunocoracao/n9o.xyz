Dockers Ziel ist es, ein erstklassiges Produkterlebnis für unsere Kunden zu schaffen. Wir wollen ein robustes Produkt entwickeln, das allen Teams hilft, ihre Ziele zu erreichen. In diesem Sinne haben wir versucht, den Prozess der Einarbeitung Ihres Teams in das Docker-Ökosystem mit unserer Funktion zum massenweisen Hinzufügen von Benutzern für Docker Business und Docker Team Abonnements zu vereinfachen.

Sie können Ihr Team zu ihren Konten einladen, indem Sie eine Datei mit ihren E-Mail-Adressen auf Docker Hub hochladen. Die CSV-Datei kann entweder eine Datei sein, die Sie speziell für diesen Zweck erstellt haben, oder eine, die aus einem anderen internen System extrahiert wurde. Die einzige Voraussetzung ist, dass die Datei eine Spalte mit den E-Mail-Adressen der Benutzer enthält, die zu Docker eingeladen werden sollen. Sobald die CSV-Datei über Docker Hub hochgeladen ist, erhält jedes Teammitglied in der Datei eine Einladung zur Nutzung seines Kontos.

<img src="/posts/202207-docker-bulk-add/image1.webp"/>

Wir haben auch die Web-Oberfläche von Docker Hub aktualisiert, um mehrere Mitglieder gleichzeitig hinzuzufügen. Wir hoffen, dass dies für kleinere Teams nützlich ist, die einfach eine Liste von E-Mails direkt in der Web-Oberfläche kopieren und einfügen und alle einarbeiten können, die sie benötigen. Sobald Ihr Team eingeladen ist, können Sie sowohl ausstehende als auch angenommene Einladungen über Docker Hub sehen.

<img src="/posts/202207-docker-bulk-add/image2.webp.webp"/>

Bulk User Add kann verwendet werden, ohne dass SSO für Ihre Organisation eingerichtet sein muss. Diese Funktion ermöglicht es Ihnen, das Beste aus Ihrem Docker Team oder Business Abonnement herauszuholen, und sie vereinfacht den Einarbeitungsprozess erheblich.

Erfahren Sie mehr über die Funktion auf unserer Dokumentationsseite und melden Sie sich bei Ihrem Docker Hub-Konto an, um es selbst auszuprobieren.
