El objetivo de Docker es crear una experiencia de producto de primera clase para nuestros clientes. Queremos construir un producto robusto que ayude a todos los equipos a alcanzar sus objetivos. En línea con eso, hemos tratado de simplificar el proceso de incorporación de tu equipo al ecosistema Docker con nuestra función de Adición Masiva de Usuarios para las suscripciones Docker Business y Docker Team.

Puedes invitar a tu equipo a sus cuentas subiendo un archivo con sus direcciones de correo electrónico a Docker Hub. El archivo CSV puede ser un archivo que crees específicamente para este propósito, o uno extraído de otro sistema interno. El único requisito es que el archivo contenga una columna con las direcciones de correo electrónico de los usuarios que serán invitados a Docker. Una vez que el archivo CSV se sube a través de Docker Hub, cada miembro del equipo en el archivo recibirá una invitación para usar su cuenta.

<img src="/posts/202207-docker-bulk-add/image1.webp"/>

También hemos actualizado la interfaz web de Docker Hub para agregar múltiples miembros a la vez. Esperamos que esto sea útil para equipos más pequeños que pueden simplemente copiar y pegar una lista de correos electrónicos directamente en la interfaz web e incorporar a todos los que necesitan. Una vez que tu equipo esté invitado, puedes ver tanto las invitaciones pendientes como las aceptadas a través de Docker Hub.

<img src="/posts/202207-docker-bulk-add/image2.webp.webp"/>

La Adición Masiva de Usuarios puede usarse sin necesidad de tener SSO configurado para tu organización. Esta función te permite sacar el máximo provecho de tu suscripción Docker Team o Business, y simplifica enormemente el proceso de incorporación.

Aprende más sobre la función en nuestra página de documentación, e inicia sesión en tu cuenta de Docker Hub para probarlo tú mismo.
