L'obiettivo di Docker è creare un'esperienza di prodotto di prima classe per i nostri clienti. Vogliamo costruire un prodotto robusto che aiuterà tutti i team a raggiungere i loro obiettivi. In linea con questo, abbiamo cercato di semplificare il processo di onboarding del tuo team nell'ecosistema Docker con la nostra funzione di Aggiunta Massiva di Utenti per gli abbonamenti Docker Business e Docker Team.

Puoi invitare il tuo team ai loro account caricando un file contenente i loro indirizzi email su Docker Hub. Il file CSV può essere un file creato appositamente per questo scopo, o uno estratto da un altro sistema interno. L'unico requisito è che il file contenga una colonna con gli indirizzi email degli utenti che saranno invitati su Docker. Una volta caricato il file CSV tramite Docker Hub, ogni membro del team nel file riceverà un invito a utilizzare il proprio account.

<img src="/posts/202207-docker-bulk-add/image1.webp"/>

Abbiamo anche aggiornato l'interfaccia web di Docker Hub per aggiungere più membri contemporaneamente. Speriamo che questo sia utile per i team più piccoli che possono semplicemente copiare e incollare un elenco di email direttamente nell'interfaccia web e fare l'onboarding di tutti quelli di cui hanno bisogno. Una volta invitato il team, puoi vedere sia gli inviti in sospeso che quelli accettati tramite Docker Hub.

<img src="/posts/202207-docker-bulk-add/image2.webp.webp"/>

L'Aggiunta Massiva di Utenti può essere utilizzata senza dover configurare SSO per la tua organizzazione. Questa funzione ti permette di ottenere il massimo dal tuo abbonamento Docker Team o Business e semplifica notevolmente il processo di onboarding.

Scopri di più sulla funzione nella nostra pagina di documentazione e accedi al tuo account Docker Hub per provarla tu stesso.
