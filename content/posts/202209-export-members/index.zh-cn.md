Docker Hub 的导出成员功能现已推出，让你能够将所有 Docker 用户的完整列表导出到单个 CSV 文件中。该文件将包含他们的用户名、全名和电子邮件地址——以及用户的当前状态和用户是否属于特定团队。如果你是管理员，这意味着你可以快速查看整个组织的 Docker 使用情况。

在成员选项卡中，你可以通过按 `Export members` 按钮下载 CSV 文件。该文件可用于验证用户状态、确认团队结构和快速审计 Docker 使用情况。

<img src="/posts/202209-export-members/image.webp"/>

导出成员功能仅适用于 Docker Business 订阅者。此功能将帮助组织更好地跟踪其 Docker 使用情况，同时简化管理员在 Docker Hub 中审查其用户所需的步骤。

在 Docker，我们持续倾听客户的声音，并努力构建使他们成功所需的工具。欢迎查看我们的<a target="_blank" href="https://github.com/docker/roadmap">公开路线图</a>并留下反馈或对更多此类功能的请求！

在我们的<a target="_blank" href="https://docs.docker.com/docker-hub/members/#invite-members">文档页面</a>了解更多关于导出用户的信息，或<a target="_blank" href="http://hub.docker.com/login">登录你的 Docker Hub 账户</a>亲自体验。
