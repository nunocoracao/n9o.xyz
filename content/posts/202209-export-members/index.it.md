La funzionalità Export Members di Docker Hub è ora disponibile, dandoti la possibilità di esportare un elenco completo di tutti i tuoi utenti Docker in un singolo file CSV. Il file conterrà il loro nome utente, nome completo e indirizzo email — così come lo stato attuale dell'utente e se l'utente appartiene a un determinato team. Se sei un amministratore, questo significa che puoi visualizzare rapidamente l'utilizzo di Docker dell'intera organizzazione.

Nella scheda Membri, puoi scaricare un file CSV premendo il pulsante `Export members`. Il file può essere utilizzato per verificare lo stato dell'utente, confermare la struttura del team e verificare rapidamente l'utilizzo di Docker.

<img src="/posts/202209-export-members/image.webp"/>

La funzione Export Members è disponibile solo per gli abbonati Docker Business. Questa funzione aiuterà le organizzazioni a tracciare meglio il loro utilizzo di Docker, semplificando anche i passaggi necessari per un amministratore per rivedere i propri utenti all'interno di Docker Hub.

In Docker, ascoltiamo continuamente i nostri clienti e ci impegniamo a costruire gli strumenti necessari per renderli di successo. Sentiti libero di consultare la nostra <a target="_blank" href="https://github.com/docker/roadmap">roadmap pubblica</a> e lasciare feedback o richieste per altre funzionalità come questa!

Scopri di più sull'esportazione degli utenti sulla nostra <a target="_blank" href="https://docs.docker.com/docker-hub/members/#invite-members">pagina di documentazione</a>, o <a target="_blank" href="http://hub.docker.com/login">accedi al tuo account Docker Hub</a> per provarlo tu stesso.
