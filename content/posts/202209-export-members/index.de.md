Die Export Members-Funktionalität von Docker Hub ist jetzt verfügbar und gibt Ihnen die Möglichkeit, eine vollständige Liste aller Ihrer Docker-Benutzer in eine einzelne CSV-Datei zu exportieren. Die Datei enthält ihren Benutzernamen, vollständigen Namen und E-Mail-Adresse — sowie den aktuellen Status des Benutzers und ob der Benutzer zu einem bestimmten Team gehört. Wenn Sie Administrator sind, bedeutet das, dass Sie schnell die gesamte Docker-Nutzung Ihrer Organisation einsehen können.

Im Mitglieder-Tab können Sie eine CSV-Datei herunterladen, indem Sie die Schaltfläche `Export members` drücken. Die Datei kann verwendet werden, um den Benutzerstatus zu überprüfen, die Teamstruktur zu bestätigen und die Docker-Nutzung schnell zu prüfen.

<img src="/posts/202209-export-members/image.webp"/>

Die Export Members-Funktion ist nur für Docker Business-Abonnenten verfügbar. Diese Funktion hilft Organisationen, ihre Nutzung von Docker besser zu verfolgen und vereinfacht gleichzeitig die Schritte, die ein Administrator benötigt, um seine Benutzer in Docker Hub zu überprüfen.

Bei Docker hören wir kontinuierlich auf unsere Kunden und bemühen uns, die Tools zu entwickeln, die sie erfolgreich machen. Schauen Sie sich gerne unsere <a target="_blank" href="https://github.com/docker/roadmap">öffentliche Roadmap</a> an und hinterlassen Sie Feedback oder Anfragen für weitere Funktionen wie diese!

Erfahren Sie mehr über das Exportieren von Benutzern auf unserer <a target="_blank" href="https://docs.docker.com/docker-hub/members/#invite-members">Dokumentationsseite</a> oder <a target="_blank" href="http://hub.docker.com/login">melden Sie sich bei Ihrem Docker Hub-Konto an</a>, um es selbst auszuprobieren.
