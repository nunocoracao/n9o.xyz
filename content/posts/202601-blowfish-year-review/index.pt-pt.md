Ao sentar-me para escrever isto no primeiro dia de 2026, dou por mim a olhar para trás para o que foi um ano extraordinário para o Blowfish. O que começou como um projeto pessoal para construir o meu próprio tema Hugo cresceu para algo que nunca imaginei - uma comunidade open-source vibrante com contribuidores de todo o mundo.

{{< github repo="nunocoracao/blowfish" >}}

## Em Números

2025 foi um ano de crescimento significativo. Eis onde estamos:

- **2.500+ estrelas** no GitHub (comparado com cerca de 600 quando escrevi o primeiro tutorial no final de 2023)
- **645 forks** do repositório
- **241 contribuidores** que ajudaram a moldar o Blowfish
- **3.928 commits** no branch principal
- **1.390 pull requests merged**
- **16 lançamentos** ao longo do ano (v2.81.0 a v2.94.0)

Estes números representam pessoas reais que escolheram usar o Blowfish para os seus sites pessoais, blogs e portfolios. Cada estrela, cada fork, cada contribuição conta a história de alguém que encontrou valor neste projeto. Isso significa o mundo para mim.

## Destaques de Funcionalidades

Lançámos muitas novas funcionalidades este ano. Aqui estão alguns destaques:

### Novos Shortcodes
- **Tabs** - Criar blocos de conteúdo em abas para melhor organização
- **Gallery** - Galerias de imagens bonitas prontas a usar
- **Gist** - Incorporar gists do GitHub diretamente no conteúdo
- **Video** - Suporte nativo para incorporação de vídeos

### Melhorias Visuais
- **GitHub Alerts/Admonitions** - Aquelas caixas de destaque que vês nos READMEs do GitHub? Agora disponíveis no Blowfish
- **Personalização completa do Chroma** - Controlo total sobre temas de syntax highlighting
- **Novos esquemas de cores** - Adicionados os esquemas "github" e "one-light"
- **Modo escuro do Mermaid** - Os diagramas agora respeitam o modo escuro do tema

### Melhorias Técnicas
- **TailwindCSS v4** - Atualização major do framework trazendo melhor performance
- **Breadcrumbs estruturados para SEO** - Otimização melhorada para motores de busca
- **Suporte para formato de imagem Base64** - Mais flexibilidade no tratamento de imagens
- **Melhorias no Medium-zoom** - Melhor experiência de zoom de imagens

### Acessibilidade e Experiência do Utilizador
- **Toggle de modo Zen** - Experiência de leitura sem distrações
- **Responder por Email** - Interação direta com leitores

### Social e Partilha
- **Suporte de partilha Bluesky** - Acompanhar a evolução do panorama social
- **Partilha Mastodon** - Opções de partilha amigáveis ao Fediverse

### Internacionalização
Este ano adicionámos suporte para várias novas línguas incluindo Indonésio, Alemão, Francês, Espanhol, Português (PT e BR), Tailandês e Ucraniano. Um grande obrigado a todos os tradutores que tornaram o Blowfish acessível a mais pessoas em todo o mundo.

## Um Momento Marcante

Um dos momentos mais significativos de 2025 foi em Outubro quando dei as boas-vindas ao [@ZhenShuo2021](https://github.com/ZhenShuo2021) como o primeiro colaborador oficial do Blowfish.

Construir um projeto open-source pode por vezes parecer uma jornada solitária. Ter alguém a dar um passo em frente para partilhar a carga de manutenção, rever PRs e ajudar a guiar a direção do projeto é inestimável. Isto marca um novo capítulo para o Blowfish - um onde o projeto é maior do que apenas uma pessoa.

## A Comunidade

Para além do código, o que torna o Blowfish especial é a sua comunidade. As GitHub Discussions tornaram-se um lugar onde os utilizadores se ajudam mutuamente, partilham os seus sites e propõem novas ideias. A secção de Q&A está ativa, com membros da comunidade a responder a perguntas antes mesmo de eu ter oportunidade de as ver.

Vi o Blowfish ser usado para blogs pessoais, portfolios de developers, sites de documentação e tudo o que há pelo meio. Cada vez que alguém partilha o seu site na secção ["Show and Tell"](https://blowfish.page/users/), lembro-me porque comecei este projeto.

## Olhando para o Futuro

À medida que entramos em 2026, estou entusiasmado com o que vem a seguir. A base que construímos é sólida, e com uma equipa crescente de contribuidores, podemos levar o Blowfish ainda mais longe. Há novas funcionalidades no roadmap, melhorias de performance a fazer, e sempre mais formas de tornar o tema mais acessível e fácil de usar.

Se estás a usar o Blowfish, adoraria ver o que construíste. Passa pelas [GitHub Discussions](https://github.com/nunocoracao/blowfish/discussions) e partilha o teu site. Se tens ideias para novas funcionalidades ou melhorias, abre uma issue ou junta-te à conversa.
